# O Ninho — Índice dos Entregáveis

Materiais do produto digital para mães brasileiras (24–45 anos) que vivem o **ciclo do grito** (explode → se culpa → jura mudar → repete), com pico às 18h.

Todos os arquivos estão em **Markdown**, prontos para virar **PDF A4**. Tom de amiga que já passou por isso — acolhedor, sem julgamento.
